Wine Quality Prediction

Classification pipeline & Interactive demo

Overview

This project builds a classification pipeline to predict wine quality from chemical properties (pH, alcohol, sulphates, etc.). The workflow includes exploratory data analysis, feature selection, model training (RandomForest / XGBoost), and a small Streamlit demo to visualize predictions and feature importance.

Key points:

  • Dataset: UCI wine quality dataset.
  • Approach: feature prep, cross-validation, model comparison, and simple calibration.
  • Result: prototype classifier with ~0.75 accuracy and an interactive demo to explore inputs.

Tech: Python, Pandas, Scikit-learn, XGBoost, Streamlit
